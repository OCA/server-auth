This module will support additional features for LDAP authentication.

- It will bypass Standard Odoo user authentication if user pattern match.

**Table of contents**

.. contents::
   :local:

Configuration
=============

- Settings > Setup Your LDAP Server > Open Record and provide user pattern
- Enable LDAP Password Change security access right to change password of LDAP user.

Usage
=====

- Set user regex pattern to disable odoo password checking mechanism
- Preferences wizard > Remove link of Password change for LDAP Users
